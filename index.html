wallmart.com

Non-empty shopping card detection

Non empty shopping cart detection in shopping centers via Top camera and Jetson Nano/Orin device

AI Problem Formulation: The project involves formulating the AI problem to address the task of multi-object tracking in challenging environments. This includes defining the objectives, identifying the relevant data sources, and determining the performance metrics for evaluating the tracking system.

Object Detection: The project utilizes object detection techniques to identify and locate objects of interest within a given frame.This involves employing deep learning models such as YOLO (You Only Look Once) or Faster R-CNN (Region-based Convolutional Neural Networks) to accurately detect objects in real-time.

Multi-Object Tracking in Challenging Environments: The project focuses on developing algorithms and methodologies for tracking multiple objects in challenging scenarios. It addresses complexities such as occlusions, scale variations, and unpredictable object movements to ensure robust tracking across frames.

Deployment and Optimization of AI Model in Edge Device: The project involves deploying and optimizing the AI model on edge devices, such as embedded systems or mobile devices, to enable real-time object tracking in resource-constrained environments. This includes techniques like model compression, quantization, and hardware acceleration to ensure efficient inference on edge devices.

By leveraging this tech stack, the project formulates the AI problem for multi-object tracking, utilizes object detection techniques for accurate object identification, develops algorithms for robust tracking in challenging environments, and deploys and optimizes the AI model on edge devices for real-time inference.

This comprehensive approach enables effective multi-object tracking in resource-constrained settings, opening up possibilities for applications in various domains, including surveillance, robotics, and autonomous systems.
